# MyTrainningSystem

穿戴式裝置設計主要透過Arduino Nano 板控制感測元件,並以 Mpu6050 Sensor(六軸感測元件)及Flex Sensor 2.2(彎曲感測元件)進行 肩部擺動及手指彎曲的數據收集,
再透過 HC-06 藍芽模組與行動裝置進行 數據上的傳輸。
